About Website

I created this website to share a bit about who I am, share resources that have helped me or that I have created, and also to practice my web development skills. I hope you like the site and find something useful in it.

Technical Details

This website is built using Flask, a lightweight web framework for Python. It uses Jinja2 templating to render HTML pages and is styled with CSS. The objective is for the site to weigh no more than 5MB in absolute size — a nod to late 2000s web design philosophy, and an excuse to keep it super fast.

  • Framework: Flask
  • Templating Engine: Jinja2
  • Styling: CSS — no frameworks, no responsive hehe
  • Container: Docker + Docker Compose
  • HTTP Server: Nginx (front-end) + Apache (back-end)
  • Website Size: < 5MB

College
